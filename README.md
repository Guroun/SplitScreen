# Split-Screen Multiplayer Mod

Play together on a single screen with a smooth and seamless split-screen experience — no complicated setup required.

## Split-Screen Layouts

Choose how you want to play:

- **Horizontal Split** — two players side by side  
- **Vertical Split** — top and bottom layout  
- **Quad Split** — up to four players on one screen  

Switch layouts instantly using a built-in visual picker (F11) with live previews.

## Seamless Multiplayer

Jump into the same world instantly:

- "Join" button on the title screen  
- Automatically connects to the server another instance is playing on  
- Works with LAN and remote servers  
- No need to enter IP addresses manually  

## Smart Audio

No more overlapping sounds:

- Proximity-based audio muting  
- When players are within 16 blocks, one instance mutes game audio  
- Music, UI sounds, and master volume are preserved  
- Fully automatic — no configuration required  

## Recommended Mods

For the best experience, it’s recommended to use optimization and controller support mods:

- **Sodium** — greatly improves performance and FPS  
- **C2ME (Concurrent Chunk Management Engine)** — enhances world loading and reduces lag  
- **Controllify** — adds full controller support for a smoother split-screen experience
